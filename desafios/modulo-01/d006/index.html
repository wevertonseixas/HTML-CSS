Tags em HTML

No HTML5, temos diversas tags que são bem uteis para nos ajudar em nosso trabalho. Como por exemplo, temos as seguintes tags (passe o mouse em cima para saber suas funções 😊):

  • Código-fonte
  • Citação símples
  • Imagens
  • Listas
  • Negrito
  • Ênfase
  • Termo deletado
  • Termo inserido
  • Abreviação
